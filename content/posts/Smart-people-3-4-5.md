# 聰明人3不管

2. 不管他人閒事。
3. 不管他人情事。

2. 胡話不說。
3. 狂話不說。
4. 假話不說。

2. 涉及錢財的忙不幫。
3. 沒辦法幫的忙不幫。
4. 不被感激的忙不幫。
5. 超出底線的忙不幫。
